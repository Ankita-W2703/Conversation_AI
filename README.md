# Hybrid RAG System (FLAN-T5)

A Hybrid Retrieval-Augmented Generation (RAG) system using:

- Dense retrieval (FAISS)
- Sparse retrieval (BM25)
- Reciprocal Rank Fusion (RRF)
- FLAN-T5 for answer generation
- Streamlit UI
- Automated evaluation & reporting

## Features

- Interactive question answering UI
- Hybrid retrieval comparison
- Automated evaluation pipeline
- PDF & HTML reports

## Run UI

```bash
streamlit run app.py

## Run Pipeline

```bash
python pipeline.py
